# 场景内移动物体实时检测系统

# 功能 ：检测视野内是否有移动物体，如果有移动物体则将画面通过4g模块发送到服务器，没有移动物体则不发送。

# 环境：树莓派，4g模块，摄像头。C/C++，Linux。
# 备注：应用于偏僻场景的实时监控。降低传输量和功耗，尽量降低延迟，能够对光干扰进行过滤。不把光干扰误识别为移动物体。
# 任务：主要完成在树莓派端的代码编写。目录结构，所需依赖项，以及下载方法。
    1. 编写代码，完成移动物体的检测。
    2. 将画面通过4g模块发送到服务器。
    3. 下载依赖项也形成具体的脚本，方便在新的树莓派上部署。
    4. 给出具体的代码实现。

    6.  给出用户从github上下载代码的方法。
    7.  给出用户编译运行的方法。
    8.  给出用户部署的方法。
    9.  给出用户测试的方法。
    10. 给出用户反馈问题的方法。
    11. 给出在服务器端接受图像并显示的代码。
    12. 给出用户如何将服务器代码部署到服务器的具体方法。
    13. 服务器的ip地址，端口号用配置文件配置，方便修改。
