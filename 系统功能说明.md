# 树莓派带摄像头检测视野内是否有移动物体，如果有移动物体则将画面通过4g模块发送到服务器，没有移动物体则不发送，用于偏僻环境的实时监控。
# 特点：
    1.  完全企业级应用规范，可直接用于企业级应用，也可以用于个人项目和实验。
    2.  降低传输量和功耗，尽量降低延迟，能够对光干扰进行过滤。不把光干扰误识别为移动物体。  
    3.  代码开源规范，文档详细，方便用户修改和定制，也有详细的测试代码。
    4.  有心跳机制，服务器端和边缘设备（树莓派）会定时发送心跳包，判断是否连接。
    5.  树莓派端和服务器端的代码都有配置文件，方便用户修改。
    6.  服务器端用python编写，便于用户加入AI算法，树莓派端用C/C++编写。
# 环境：树莓派，4g模块，摄像头。C/C++，Linux。
